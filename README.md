# Ticket Management System

## Project Overview

A lightweight internal Ticket Management System frontend built with React and TypeScript.

The application allows admins to:

- View tickets
- Create tickets
- Track ticket status
- Manage ticket priority
- Organize tickets across multiple domains

The project focuses primarily on the frontend implementation, UI structure, reusable components, and clean user experience.

---

## Assumptions & Limitations

- The project primarily focuses on the frontend implementation.
- Table pagination is not implemented.
- The application is not optimized for extremely large datasets (e.g. 10k+ tickets).
- Mobile responsiveness was not implemented since responsive design was not explicitly mentioned in the assignment requirements.
